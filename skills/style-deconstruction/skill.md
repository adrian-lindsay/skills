# Content Style Deconstruction & Replication Skill

## Purpose
Extract and reverse-engineer the psychological, structural, and linguistic writing style of any text sample.

## Core Capability
This skill analyzes:
- Surface patterns (formatting, punctuation, vocabulary, structure)
- Psychological architecture (persuasion frameworks and emotional triggers)
- Subconscious rhythm (pacing, flow, cadence)
- Linguistic fingerprints (syntax and structural preferences)
- Influence mechanisms (engagement and memorability drivers)
- Replication blueprint (rules for recreating the style)

## Input
Any written content:
- Blog posts
- Emails
- Social posts
- Sales copy
- Scripts
- Long-form content

## Output
A structured style profile and replication framework that allows near-indistinguishable voice recreation.

## Methodology
1. Surface Pattern Analysis
2. Psychological Architecture Mapping
3. Rhythm & Flow Analysis
4. Linguistic Fingerprinting
5. Influence Mechanism Identification
6. Replication Blueprint Creation
